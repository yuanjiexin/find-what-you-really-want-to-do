# 找到真正想做的事 —— Claude Skill

基于八木仁平《找到你真正想做的事》的自我理解方法，实现为 Claude Skill。

Claude 会以教练的身份全程陪伴，通过对话帮你找到并说清楚「真正想做的事」。

## 核心公式

```
喜欢的事（热情）× 擅长的事（才能）× 重要的事（价值观）＝ 真正想做的事
```

## 适合谁用？

- 知道自己「想做点什么」，但说不清楚是什么
- 读完书但做不下去里面的练习
- 自己想总是想不明白，需要有人引导
- 正在考虑换工作、考研、创业，但方向不清晰
- 感觉每天都在应付，找不到动力的根源

## 安装方法

### 方式一：使用 .skill 文件（推荐）

1. 从 [Releases](../../releases) 下载最新的 `.skill` 文件
2. 在 Claude.ai 设置 → Skills → 导入，选择该文件

### 方式二：手动安装

1. 下载或克隆本仓库
2. 将整个文件夹放入你的 Skills 目录

## 怎么用？

安装后，直接跟 Claude 说就行：

> 「我不知道自己想做什么，帮我想想」  
> 「我对职业方向很迷茫」  
> 「想做自我分析」  
> 「用八木仁平的方法帮我找到想做的事」

Claude 会自动启动这个技能，开始对话式的引导流程。

## 三个步骤

**第一步** — 找到「重要的事」（价值观）  
锁定「为什么工作、为什么活着」的根基。

**第二步** — 找到「擅长的事」（才能）  
把你无意识中就在用的思维和行动方式说清楚。

**第三步** — 找到「喜欢的事」（热情）  
确定你感兴趣的领域和主题。

**整合** — 三者相乘  
用一句话说出「真正想做的事」。
